An AI-powered PDF chatbot using RAG, LangChain, FAISS, HuggingFace Embeddings, and Llama 3.2 for intelligent document analysis.
# 📄 IntelliDocs AI

## AI-Powered PDF Chatbot using RAG

IntelliDocs AI is an intelligent document assistant that allows users to upload PDF files and ask questions using natural language.

The application uses Retrieval-Augmented Generation (RAG) to retrieve relevant information from documents and generate accurate answers using a local Large Language Model.

---

# 🚀 Features

✅ Upload PDF documents  
✅ Extract text from PDFs  
✅ Intelligent document chunking  
✅ Semantic search using vector embeddings  
✅ FAISS vector database for fast retrieval  
✅ AI-powered question answering  
✅ ChatGPT-style conversation interface  
✅ Source references for generated answers  
✅ Runs locally using Llama 3.2
